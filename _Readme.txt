## 操作説明
----------------------------------------------------------------------------
《マウス》
左クリック … 決定、ボタンを押す
右クリック … キャンセル(オプションのみ)

《キーボード》
Space,Z … 決定
Enter … 解答
Shift,X … キャンセル
←↑↓→ … 方向、決定
F2 … FPS、ms表示
F4 … フルスクリーン(DL版のみ)
F5 … 更新、再起動

解答には、A～Zまでのキーを使用します。
